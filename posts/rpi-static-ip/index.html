라즈베리 파이의 경우 내부 DHCP 쪽 설정을 통해 고정 IP를 설정할 수 있다.

우선 다음과 같이 입력해 Nano로 다음 파일을 연다.

sudo nano /etc/dhcpcd.conf

파일 중간에 다음과 같은 내용이 있는 부분이 있다. 이 부분의 주석을 해제한다. 여기서 ip_address에 원하는 IP 주소를 입력하면 된다.

interface eth0
static ip_address=192.168.0.10/24
static routers=192.168.0.1
static domain_name_servers=192.168.0.1
